# guice-freemarker-servlet
Guice / Freemarker / Servlet

minimum standalone web app example by Java

## Architecture Base

* Java 17
* Jetty 11.0.x
* Guice/Guice-Servlet 7.0.0
* MicroProfile Config/Apache Geronimo
* Gson 2.11.x
* FreeMarker 2.3.33
* Log4j2/Slf4j

### build

```sh
cd app
mvn clean package
java -jar target/app-1.0.0.jar
```
